Welcome to the Hawaii Climate Measurement API!

Below you will find the api's available for this website:

  • /api/v1.0/precipitation
    • Endpoint returns the dates and precipitation as key value pairs.
    • [{"2015-01-01": 0.23}]
  • /api/v1.0/stations
    • Endpoint returns Measurement Stations
  • /api/v1.0/tobs
    • Endpoint returns temperature observations for the past year
  • /api/v1.0/<start>
    • Endpoint returns the following for all dates since the start (inclusive):
      • TMIN: Min Temperature
      • TAVG: Avg Temperature
      • TMAX: Max Temperature
    • e.g. /api/v1.0/2015-01-01
  • /api/v1.0/<start>/<end>
    • Endpoint returns the following for all dates since the start (inclusive) but before the end (inclusive):
      • TMIN: Min Temperature
      • TAVG: Avg Temperature
      • TMAX: Max Temperature
    • e.g. /api/v1.0/2015-01-01/2016-02-03
